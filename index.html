Admin & IAM

Controlled &
Secure Access

We avoid direct server access for maximum security.

🚫 No SSH — All access via SSM

🖥️ AWS Systems Manager (SSM) — browser shell
🚫 No SSH ports open — attack surface minimized
👤 IAM Role for EC2 — principle of least privilege
🔑 Read Secrets — from Secrets Manager only
📝 Write Logs — to CloudWatch log groups
📧 Send Email — via SES for notifications
